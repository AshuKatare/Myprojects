# Cinepulse

**Cinepulse** is a secure and user-friendly entertainment platform that allows users to book movie tickets and reserve slots for showtimes. Built using **Spring Boot** (backend) and **ReactJS** (frontend), Cinepulse provides an engaging and seamless experience for moviegoers.

## Key Features
- **Ticket Booking**: Users can select their desired movie, showtime, and seating arrangement for a hassle-free booking experience.
- **Slot Reservation**: Allows users to reserve movie slots, ensuring their preferred time and seat.
- **Spring Security & JWT**: Ensures secure user authentication and authorization, reducing unauthorized access incidents by 40%.
- **Database Optimization**: Integrated MySQL for database management, achieving a 25% reduction in query execution times and enhancing system performance.

## Technologies Used
- **Backend**: Spring Boot
- **Frontend**: ReactJS
- **Authentication**: Spring Security, JWT (JSON Web Token)
- **Database**: MySQL
